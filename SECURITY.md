# Security Policy

## Supported versions

Security fixes are prioritized for the latest `master` branch.

## Reporting a vulnerability

Please report vulnerabilities privately by emailing: `security@gitweaver.dev`.

Include:

- A clear description of impact.
- Reproduction steps or proof-of-concept.
- Suggested remediation, if known.

We will acknowledge receipt within 3 business days and provide remediation status updates.

## Disclosure

Please do not open public issues for unpatched vulnerabilities.
